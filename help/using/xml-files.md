# File XML {#xml-files}

I siti Web che utilizzano un sistema basato su XML per gestire immagini e informazioni sulle immagini possono caricare file XML in Adobe Dynamic Media Classic. Potete designare uno di questi file come file del set di regole di preelaborazione per Image Server. Questo file ristruttura il formato di protocollo Image Server standard per soddisfare la logica di lavoro del server. Nella schermata Configurazione Ã¨ possibile specificare un file XML da utilizzare come percorso del file delle definizioni del set di regole. Questa impostazione del percorso si trova in Gestione catalogo nella schermata Pubblicazione su Server immagini. Consultate [Server immagini](publish-setup.md#image_server).
